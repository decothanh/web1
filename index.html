Hướng dẫn sử dụng Gallery

Để thêm ảnh từ Google Photos:

  1. Mở Google Photos album của bạn
  2. Click vào từng ảnh → Share → Copy link
  3. Thay thế URL trong code JavaScript bên dưới
  4. Thêm thông tin title, category cho mỗi ảnh

Để nhúng vào Google Sites: Copy toàn bộ code HTML này và paste vào Google Sites bằng cách chọn Insert → Embed → Custom HTML
